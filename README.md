## Scenic Area Ticket Booking System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/05cd3876fe09430d86091c8ab3b07730/ghb20250916) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Forum
5. Attraction Information
6. Announcement Information
7. Personal Center
8. Attraction Favorites
9. Tourist Attraction Booking
10. Change Password
11. Personal Information
12. Forum Management
13. Attraction Message Management
14. Attraction Favorites Management
15. Tourist Attraction Booking Management
16. Announcement Information Management


**Administrator Functions:**
1. Login
2. Homepage
3. Personal Center (Change Password, Modify Information)
4. Administrator Management
5. Attraction Type Management
6. Announcement Type Management
7. Forum Management
8. Attraction Information Management
9. Attraction Message Management
10. Attraction Favorites Management
11. Tourist Attraction Booking Management
12. Announcement Information Management
13. User Management
14. Carousel Management
